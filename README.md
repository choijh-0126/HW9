# HW9

주요기능

1. GameMode 서버 로직 구현
1) 서버에서 전체 게임 흐름 관리
2) 플레이어별 상태 추적

2. 난수 생성 로직
1) 랜덤한 숫자 생성하여 게임의 비밀 숫자 역할 수행

3. 판정 로직
1) 플레이어가 입력한 숫자와 정답 숫자를 비교
2) 스트라이크, 볼, 아웃 판정 처리
  
4. 시도 횟수 및 상태 관리
1) 플레이어의 입력 기록 및 남은 시도 횟수 추적
2) 게임 진행 상황 UI 및 채팅 반영

5. 승리, 무승부, 게임 리셋
1) 정답 맞춘 경우 즉시 승리 처리
2) 모든 플레이어 실패 시 무승부 판정
3) 새로운 게임 시작 가능
